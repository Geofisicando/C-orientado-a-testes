#### Exemplo exibir mensagem colorida do Unity

Para habilitar a saída colorida das mensagens de erro no Unity, acrescente a
seguinte linha na linha 10 em Unity/unity.h:

```c
#define UNITY_OUTPUT_COLOR
```
